
# Personal Finance Analyzer

A C++ console-based application that helps users track income, expenses, 
and savings goals with persistent file storage.

## Features
- Add and delete income and expense entries
- View monthly summaries
- Save and load data from files
- Input validation to prevent errors

## Technologies Used
- C++
- File I/O
- Object-Oriented Programming

## What I Learned
- Designing structured C++ programs
- Working with files for data persistence
- Debugging and handling edge cases

## Future Improvements
- Add category-based expense analysis
- Generate visual spending reports
- Migrate to a web-based interface
